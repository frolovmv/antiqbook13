Значит так:

1.
http://web-engine.ruweb.net/f_catalog/admin.php
Админу. Файлы с csv и всяки-разны каталоги должны заливаться в ./папка со скриптом/data/ 
Файлы с расширением csv автоматически показываются в админпанели.
Однако если под файлом csv будет скрываться файл, не являющийся csv - скрипт при парсинге базы даст отказ всем невалидным строкам.
 
Для тестовой ситуации пароль admin/admin, который легко меняется в настройках
(cfg/main.cfg, должен быть защищён правами запрещающими чтение из 
браузера (0770) + файл (уже существует) .htaccess запрещающий доступ через веб-сервер Апач...). 
Также в этом файле указывается имя пользователя , пароль и имя базы МуСКЛ.
 
2.
http://web-engine.ruweb.net/f_catalog/catalog.php?part=farfor
 
Где part имя отпарсеной базы.
 
Пример поиска в этой базе . Запрос "Зураб"
http://web-engine.ruweb.net/f_catalog/catalog.php?find=%C7%F3%F0%E0%E1&find_fc=1&part=farfor&page=0
 
Конечно же можно и нужно искать через форму.
